Faceball - Sports Highlights App (React Native & Firebase)

Faceball is a sports highlights app built using React Native and Firebase that provides real-time sports video highlights, live scores, and the latest sports news. This app is designed for sports enthusiasts who want to stay updated with all the exciting moments in their favorite games.

Key Features:

Real-time Sports Video Highlights: Watch highlights of recent games and memorable plays.
Live Scores: Stay updated with live scores from various sports events.
News Feed: Get the latest sports news and updates right on your home screen.
Firebase Integration: Firebase is used for authentication, database management, and real-time data syncing.
Tech Stack:

React Native: Cross-platform mobile app development.
Firebase: Backend for authentication, real-time database, and cloud functions.
React Navigation: Navigation setup for a smooth user experience.
Axios: API requests for fetching live data.
React Native Video: For playing video highlights.
React Native Paper: UI components for an elegant design.
