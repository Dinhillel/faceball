Name;FACEBALL ⚽️🏀🏈⚾️🥎🎾
A React Native application delivering real-time sports video highlights and summaries. The app utilizes Firebase (Authentication, Firestore, Storage) for user management, data storage, and dynamic content. State management is handled using Redux Toolkit.

The UI is optimized for both Android and iOS platforms, following their respective design guidelines. React Navigation provides smooth and intuitive navigation between screens. Additionally, the app supports multiple languages through i18next localization.

Key technologies:

React Native 0.80

Firebase (Auth, Firestore, Storage)

Redux Toolkit for state management

React Navigation (Stack and Bottom Tabs)

i18next for localization support

UI optimized for both Android and iOS
